## 1. 产品概述

废旧家电回收估价平台，面向居民、回收员和站点管理员三类角色，实现家电品类估价、上门预约与回收范围管理。通过本地持久化与业务规则校验，保障估价流程规范、预约唯一可追溯。

- 解决居民旧家电处置信息不对称、估价不透明的痛点
- 目标用户：社区居民、回收员、回收站点管理员

## 2. 核心功能

### 2.1 用户角色

| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 居民 | 首页选择"居民入口" | 选择品类、填写成色、查看估价、预约上门 |
| 回收员 | 首页选择"回收员入口" | 查看估价列表、管理预约、标记处理状态 |
| 站点管理员 | 首页选择"管理员入口" | 维护回收品类与范围、配置危险拆解标记 |

### 2.2 功能模块

1. **居民估价页**：品类选择、成色填写、估价计算、上门预约
2. **回收员工作台**：估价列表、预约管理、状态流转
3. **管理员后台**：品类管理、回收范围配置、危险拆解标记

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 角色入口 | 三角色入口导航，品牌视觉展示 |
| 居民估价页 | 品类选择 | 展示可回收品类卡片，支持搜索筛选，危险拆解品类标注"不可上门" |
| 居民估价页 | 成色填写 | 成色等级选择（全新/九成新/七成新/五成新/无法使用），成色缺失时估价按钮不可点击并显示提示 |
| 居民估价页 | 估价结果 | 根据品类×成色计算估价金额，展示估价明细 |
| 居民估价页 | 上门预约 | 填写地址与预约时间，同地址同日只保留一个预约，危险拆解品类不可选择上门 |
| 回收员工作台 | 估价列表 | 查看所有居民提交的估价记录，按状态筛选 |
| 回收员工作台 | 预约管理 | 查看预约详情，标记已处理/已取消 |
| 管理员后台 | 品类管理 | 新增/编辑/删除回收品类，设置估价基数，标记是否危险拆解 |
| 管理员后台 | 回收范围 | 配置服务区域与上门范围 |

## 3. 核心流程

居民通过选择家电品类和填写成色获取估价，若品类为危险拆解类型则不支持上门回收，成色未填写则无法估价。提交预约时校验同地址同日唯一性，回收员查看并处理预约。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深森林绿 `#1B4332` 搭配暖琥珀色 `#E9A319` 点缀，传达环保回收与温暖信赖
- **辅助色**：浅灰绿 `#D8F3DC` 作背景，白色卡片承载内容
- **按钮风格**：圆角 `rounded-xl`，主按钮填充绿色，危险/警告按钮琥珀色
- **字体**：标题使用 `Noto Sans SC` 粗体，正文使用 `Noto Sans SC` 常规
- **布局风格**：左侧导航 + 右侧内容区，卡片式布局，圆角阴影
- **图标**：lucide-react 图标库，绿色系图标搭配琥珀色警告标记

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | 角色入口 | 全屏绿色渐变背景，三列卡片入口，悬停放大动画 |
| 居民估价页 | 品类选择 | 网格卡片布局，品类图标+名称，危险拆解品类显示红色角标 |
| 居民估价页 | 成色填写 | 五级成色单选按钮组，选中高亮，未选时估价按钮灰色禁用+提示文字 |
| 居民估价页 | 估价结果 | 居中大号金额展示，细目分列，预约按钮 |
| 居民估价页 | 上门预约 | 地址输入+日期选择，不可上门品类隐藏此区域 |
| 回收员工作台 | 估价列表 | 表格+状态标签，筛选栏 |
| 回收员工作台 | 预约管理 | 卡片列表，状态流转按钮 |
| 管理员后台 | 品类管理 | 左树右表，新增/编辑弹窗 |
| 管理员后台 | 回收范围 | 区域多选，地图式选择器（简化为列表） |

### 4.3 响应式设计

- 桌面优先设计，最小宽度 1024px
- 平板端（768-1024px）缩小间距，两列布局
- 移动端（<768px）单列堆叠，底部导航替代侧栏

## 5. 业务规则

| 规则编号 | 规则描述 | 触发场景 |
|----------|----------|----------|
| BR-01 | 危险拆解品类不可上门回收 | 居民选择品类后，若品类标记为危险拆解，隐藏上门预约区域 |
| BR-02 | 成色缺失不能估价 | 成色未选择时，估价按钮禁用，下方显示"请先选择成色等级"提示 |
| BR-03 | 同一地址同日只保留一个预约 | 提交预约时校验地址+日期组合唯一性，重复则弹出提示 |
